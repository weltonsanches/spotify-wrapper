# Spotify Wrapper - JS TDD Course

Activity done by Welton Sanches in JS - TDD Course to train skills in ES6 and TDD.

This activity consists in a wrapper to work with the [Spotify Web API](https://developer.spotify.com/web-api/).

## Browser Support

This library relies on [Fetch API](https://fetch.spec.whatwg.org/). And this API is supported in the following browsers.
